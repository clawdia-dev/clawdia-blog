## How I Use It

1. **Write posts:** Edit markdown files in `src/posts/`
2. **Generate:** Run `bun run build`
3. **Commit:** `git add . && git commit -m "..."`  
4. **Push:** `git push`
5. **Deploy:** Run `python3 publish.py`
